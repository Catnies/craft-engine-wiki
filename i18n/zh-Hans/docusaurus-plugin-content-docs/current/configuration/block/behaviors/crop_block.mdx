---
title: 🌽 农作物方块
id: crop_block
---

| 属性名称 | 属性类型 | 是否必需 |
|------|------|------|
| age  | int  | 是    |

**农作物方块**是一种会随时间生长的方块类型。当接收到随机刻时，它有概率进入下一生长阶段。但若光照不足，农作物将停止生长或掉落为物品。

```yml
blocks:
  default:ender_pearl_flower:
    behavior:
      type: crop_block
      grow-speed: 0.25  # 0~1 随机刻生长概率
      light-requirement: 9  # 该农作物需要的光照条件
      is-bone-meal-target: true # 默认 true
      bone-meal-age-bonus:
        type: uniform
        min: 1
        max: 3
```

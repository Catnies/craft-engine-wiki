---
title: 🍁 树叶方块
id: leaves_block
---

| 属性名称        | 属性类型    | 是否必需 |
|-------------|---------|------|
| distance    | int     | 是    |
| persistent  | boolean | 是    |
| waterlogged | boolean | 否    |

**树叶方块**是与树干分离后会自然枯萎的方块。当 `persistent` 设为 true 时将永远不会枯萎。`distance` 属性决定树叶允许离树干的最大距离。

```yml
blocks:
  default:palm_leaves:
    behavior:
      type: leaves_block
```

![](/img/leaves_block.png)

:::tip

可根据需要调整 distance 数值。对于特别大型的树木，为防止树叶枯萎可将 distance 设为 10 或更高。原版游戏中树叶最大距离为 7。

:::

---
title: 🚟 悬挂方块
id: hanging_block
---

**悬挂方块**是一种必须附着在其他方块下方的方块类型。当上方支撑方块被破坏时，悬挂方块也会随之破坏。

```yml
blocks:
  default:stalactites:
    behavior:
      type: hanging_block
      stackable: false
      blacklist: false # 使用黑名单模式
      above-block-tags: []
      above-blocks: []
      delay: 0
```

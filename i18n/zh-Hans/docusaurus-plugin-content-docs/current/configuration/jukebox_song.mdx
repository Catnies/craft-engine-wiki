---
title: 💽 唱片机曲目
id: jukebox_song
---

:::caution

由于 Minecraft 的注册表在注册后不可变，因此你需要重启服务器才能应用新的修改。不过，如果你在配置中修改了曲目的命名空间ID，则可以实时注册新的曲目，无需重启服务器。

:::

```yaml
jukebox-songs:
  default:credits_music:
    sound: minecraft:music.credits
    length: 100.0  # 音乐时长（秒）
    description: "Credits"  
    comparator-output: 15
    range: 32
```

只需添加 `minecraft:jukebox_playable` 组件，即可使物品变为可播放的音乐唱片

```yaml
items:
  default:music_stick:
    material: stick
    data:
      jukebox-playable: default:credits_music
```

如需了解如何自定义音效，请参阅[此页面](sound.mdx)。

## TTF

<UrlCard
  url="https://zh.minecraft.wiki/w/自定义字体#ttf"
  title="TrueType矢量字体"
/>

对于 TTF 字体，你需要在以下路径中创建一个 `default.json` 文件。如果你已经有一个 `default.json` 文件，只需将你的字体 JSON 追加到现有 JSON 文件的末尾即可。

<div style={{textAlign: 'center'}}>
  <img src={require('/img/font_1.png').default} alt="" />
</div>

```json
{
    "providers": [
        {
            "type": "ttf",
            "file": "minecraft:custom_font.ttf",
            "oversample": 10,
            "size": 11
        }
    ]
}
```

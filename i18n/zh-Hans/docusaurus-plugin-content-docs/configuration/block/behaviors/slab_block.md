---
title: ➖️ Slab Block
id: slab_block
---

| Property Name | Property Type | Required |
|---------|---------|---------|
| type    | slab_type   | yes   |
| half    | single_block_half   | yes   |
| waterlogged    | boolean   | no   |

```yml
blocks:
  default:palm_slab:
     behavior:
       type: slab_block
```

![](/img/slab_block.png)

:::tip

You can create at most 5 new slabs

:::
